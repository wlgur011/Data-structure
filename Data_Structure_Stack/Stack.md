# Data-structure

# stack

-----------------------

```
stack : 자료의 입출력이 후입선출의 형태로 일어나는 자료구조

stack 정리

1. 후입선출 (Last In First Out) 방식
2. 스택의 중간에서는 데이터의 삽입과 삭제가 일어날 수 없다.


```

# method

-----------------------

```

Init () : 스택의 초기화
is_empty () : 스택이 비어있는지 확인
is_full () : 스택이 가득찼는지 확인
size() : 스택의 사이즈 확인
push(x) : x를 스택의 맨 위에 추가
pop() : 맨 위의 요소를 삭제후 반환
peek(): 맨 위의 요소를 반환

```
